# Knowledge Bundles Index

Bundle 폴더의 역할·배치 규칙은 [README](./README.md)를 따른다. 이 문서는 Obsidian 탐색용 목록이다.

## 도메인

- `company/` — 전사 공통 지식
- `product/` — 제품 지식
- `engineering/` — 개발·시스템 지식
- `cs/` — 고객 지원 지식
- `operations/` — 운영 지식
- `marketing/` — 마케팅 지식

각 도메인의 실행 가능한 절차는 `runbooks/` 하위의 `type: runbook`, `status: active` Bundle만 사용한다.
Draft 후보는 기본 탐색·질의·실행 대상이 아니다.
